Argon + A-Frame Examples

The A-Frame javascript library lets you create WebVR applications using custom DOM elements, allowing 3D content to be manipulated similarly to 2D HTML elements.

We have adapted A-Frame to work with Argon, creating a new ar-scene element and a number of entities and components to support AR. The documentation and source are on the github page.
